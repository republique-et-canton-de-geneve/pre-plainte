# 📚 Documentation projet

Ce module contient la documentation technique du projet.

## Structure

- `guides/` : guides pratiques (ex: ajout de champ)
- `architecture/` : documentation architecture (à venir)
- `api/` : documentation API (Swagger / OpenAPI) (à venir)
- `adr/` : décisions d’architecture

## Objectif

Centraliser la connaissance technique et faciliter :
- l’onboarding
- la maintenance
- l’évolution du projet
